# Tic Tac Toe with Firebase Authentication

A web-based Tic Tac Toe game with user authentication and stat tracking.

## Features
- Firebase Authentication (Email/Password)
- Cloud Firestore for storing user statistics
- Single Player vs Computer
- Two Player mode
- Win/Loss/Draw tracking

## How to Play
1. Open `index.html` in your browser
2. Sign up or login
3. Play and track your stats!

## Technologies
- HTML5, CSS3, JavaScript
- Firebase Authentication
- Cloud Firestore
